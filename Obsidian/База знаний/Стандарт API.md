#Техника #Авто 
[[Авто масла]]

|         |                                                                                                                                                                         |                                                                            |
| ------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| **SQ**  | Масла последнего поколения, обеспечивающие повышенную защиту двигателя, улучшенную совместимость с уплотнениями и соответствие самым современным экологическим нормам.  | Для самых современных автомобилей, выпущенных после марта 2025 года.       |
| SP      | Масла, актуальные для современных бензиновых моторов с турбонаддувом и фильтрами выхлопа.                                                                               | Для автомобилей, выпущенных с мая 2020 года.                               |
| SN      | Адаптированы к уплотнителям разных типов, снижено содержание фосфора. Подходят для машин с системами нейтрализации отходящих газов и турбонаддувом.                     | Используются в моделях с 2011 года выпуска и позднее.                      |
| SN PLUS | Отличается от API SN дополнительными требованиями для защиты двигателя от эффекта  LSPI («Low Speed Pre-Ignition» – «Преждевременное воспламенение на малых оборотах»). |                                                                            |
| SM      | Поддерживают стабильную работу турбированных и многоклапанных моторов. Повышенная устойчивость к окислению, улучшенные низкотемпературные свойства.                     | Подходят для автомобилей, произведенных с 2004 по 2010 годы.               |
| SL      | Эффективны при работе на обедненной смеси, снижают расход топлива.                                                                                                      | Рекомендуются для машин, выпущенных до 2004 года включительно.             |
| SJ      | Универсальное масло с базовым набором защитных свойств. Может применяться в бензиновых моторах без современных требований по выхлопу.                                   | Допускается к использованию в двигателях, выпущенных не позднее 1996 года. |
